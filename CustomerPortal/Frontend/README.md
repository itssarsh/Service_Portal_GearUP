# Customer Frontend

Customer frontend ka actual app-style code ab is folder ke andar available hai.

- `src/App.js` customer app routes
- `src/portals/customer/pages/` customer UI screens
- `src/portals/customer/services/api.js` customer APIs
- `src/portals/customer/utils/session.js` customer session handling
- `src/components/` shared UI providers used by customer app

Main project ka existing frontend bhi abhi alag se present hai, lekin customer
frontend ka dedicated code structure ab yahan bhi mil jayega.
