# Customer Portal

Standalone customer app (separate from the `../mechanic` app).

- Backend: `./backend` (defaults to port `5001` to avoid clashing with mechanic backend on `5000`)
- Frontend: `./frontend`

## Run

Backend:

- `cd backend && npm install && npm start`

Frontend:

- `cd frontend && npm install && npm start`

Customer API base: `http://localhost:5001/api/customer`
