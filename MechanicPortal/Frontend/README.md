# Mechanic Frontend

Mechanic frontend ka actual app-style code ab is folder ke andar available hai.

- `src/App.js` mechanic app routes
- `src/portals/mechanic/pages/` mechanic UI screens
- `src/portals/mechanic/services/api.js` mechanic APIs
- `src/portals/mechanic/utils/session.js` mechanic session handling
- `src/components/` shared UI providers used by mechanic app

Main project ka existing frontend bhi abhi alag se present hai, lekin mechanic
frontend ka dedicated code structure ab yahan bhi mil jayega.
